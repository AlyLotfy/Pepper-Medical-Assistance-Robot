How to Run for the Demo
Without a SLAM map (simpler, works right away):
nav_bridge.py uses ALMotion.moveTo(x, y, theta) — Pepper moves relative to where it's standing. Edit navigation_targets.json coordinates to match your room layout.
With a SLAM map (full obstacle avoidance):
Run map_exploration.py once to scan the room → saves prototype_room.explo
nav_bridge.py auto-detects the map and uses navigateToInMap() instead
Prototype room stations (in navigation_targets.json):
Station A — 1.5m forward
Station B — 1.5m left, facing 90°
Station C — 1.2m forward + 1.0m left, diagonal
Reception — returns to start [0, 0, 0]
